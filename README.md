# ⚾ 야구9단 매니저

야구 팀 운영 시뮬레이션 — 로스터 구성, 경기 지휘, 감독 체험

## 🚀 GitHub Pages 배포

파일을 레포에 그대로 업로드하고 Settings → Pages → main 브랜치 선택

```
index.html      ← 메인 게임
manifest.json   ← PWA 설정
sw.js           ← 오프라인 캐시
icon-192.png    ← 앱 아이콘 (소)
icon-512.png    ← 앱 아이콘 (대)
```

접속: `https://[유저명].github.io/[레포명]/`

## 📱 PWA 설치

모바일에서 접속 → "홈 화면에 추가" → 앱처럼 실행

## 🎮 주요 기능

- 🏟️ **포메이션 뷰** — 감독 시점 수비 배치도 실시간 표시
- 🔄 **교체 시스템** — 투수교체 / 대타 기용 / 수비 포지션 변경
- 💬 **선수 반응** — 경기 상황별 불만·긍정 피드백 실시간 출력
- ⚾ **경기 중계** — 픽셀 야구장, 이닝별 라인스코어, 중계 로그
- 👥 **로스터 관리** — 영입·방출·훈련·계약 연장
- 💰 **FA·트레이드·드래프트** — 선수 수급 전략
- 📂 **CSV 선수 임포트** — 직접 만든 선수 데이터 불러오기
